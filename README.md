# Inventory System
**Playmatrix Studio** — Unity 6.3 LTS | Plug-in-ready

A complete drag-and-drop inventory system with item stacking, tooltips, rarity colours, a hotbar, world item drops, and auto-pickup — all driven by ScriptableObjects.

## Features
- 24-slot grid inventory + 5-slot hotbar (hotbar fills first)
- Drag and drop items between slots
- Stack splitting: right-click (PC) / long press 0.5s (mobile)
- Item tooltips with rarity colour coding
- World item drop and auto-pickup
- Mobile on-screen controls (joystick + BAG button, auto-shown on Android)
- PC, Android (Landscape Left), WebGL — all tested
